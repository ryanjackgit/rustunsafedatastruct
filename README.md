# rustunsafedatastruct
本项目主要是为了理解Rust语言底层使用unsafe来实现一些重要的数据结构，尤其是理解裸指针的使用，未定义行为的定义及产生.
  
实现的数据结构：

Vector

双向list 

Binary tree 

Binary search tree

Heap 

hashmap 

QuickSort

Binary search

以及双向list的iterator into_iter的引用，可变引用，by value的实现


本项目适合于对Rust语言有兴趣且有一定Rust编写经验的人学习标准库中常用数据结构的实现，对初学者也有很好的指导作用
